# 🛒 TechnoMart - Интернет-магазин электроники

- ✅ **Золотое сечение** в пропорциях дизайна

## 🚀 **Быстрый старт**

### **1. Клонирование и установка**

```bash
# Клонируйте репозиторий
git clone <repository-url>
cd next

# Установите зависимости
npm install
# или
yarn install
# или
pnpm install
```

### **2. Настройка окружения**

Создайте файл `.env.local`:

```env
# База данных
DATABASE_URL="postgresql://username:password@localhost:5432/technomart"
DIRECT_URL="postgresql://username:password@localhost:5432/technomart"

# NextAuth
NEXTAUTH_SECRET="your-secret-key-here"
NEXTAUTH_URL="http://localhost:3000"

# Загрузка файлов (опционально)
UPLOADTHING_SECRET="your-uploadthing-secret"
UPLOADTHING_APP_ID="your-uploadthing-app-id"
```

### **3. Настройка базы данных**

```bash
# Примените миграции
npx prisma db push

# Заполните тестовыми данными
npm run db:seed

# Откройте Prisma Studio (опционально)
npm run db:studio
```

### **4. Запуск проекта**

```bash
# Режим разработки
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшн версии
npm run start
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## 🎨 **Дизайн-система**

### **Цветовая палитра**
```css
/* Основные цвета (максимум 3) */
--primary: #3b82f6    /* Синий */
--secondary: #64748b  /* Серый */
--accent: #f59e0b     /* Оранжевый */
```

### **Типографика**
- **Display**: Inter (заголовки)
- **Body**: System UI (основной текст)
- **Accent**: Montserrat (кнопки, акценты)

### **Размеры по золотому сечению**
- `text-2xl`: 1.618rem (~26px)
- `text-3xl`: 2.618rem (~42px)
- `golden`: 1.618rem отступы
